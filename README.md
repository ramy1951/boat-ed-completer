# Boat Ed Helper
A program that works on **boat-ed.com**
The program will log you in and start where you finished last and keep clicking next until it hits a quiz screen or you exit the program.

### Saves you the wait timer and don't worry the question are on google anyways.

## Steps:

1. Provide login in credentials in varibles `username` and `password`.
2. Run `npm install`
3. Run the program using `node app.js`

### Note the program will NOT stop on its own. To stop the program hit `^c`.
